Minechem Development server

The following modpack is running on the Minechem development server at "minechem.jakimfett.com".
The dev server is whitelisted and is used for testing alpha/beta builds of Minechem with various mods.

When possible, I've linked to the mod page and creator's Patreon page.

This pack is on the FTB Launcher (code is 'minechem').
You can download (even more) unstable versions of the pack from this link.
