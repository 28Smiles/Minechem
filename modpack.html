ChemLab (Minechem Dev Modpack) Server

The ChemLab modpack is running on my development server at "minechem.jakimfett.com".
The dev server is whitelisted and is used for testing alpha/beta builds of various mods.

When possible, I've linked to the mod page and creator's Patreon page.

This pack is on the FTB Launcher (code is 'minechem'), but isn't always updated.
You can download up-to-date (but sometimes unstable) versions of the pack from the build server.
